# 🍔 Food Delivery MERN App

A full-stack food delivery web application built using MERN Stack.

## 🚀 Live Demo

Frontend:
https://food-delivery-client-7te8.onrender.com

Backend:
https://fooddelivery-mern.onrender.com

## ✨ Features

- User Registration & Login
- JWT Authentication
- Food Listing
- Food Details
- Add to Cart
- Favorites
- Order Management

## 🛠️ Technologies Used

### Frontend
- React.js
- Axios
- React Router
- Material UI

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
